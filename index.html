How to play:

Memory is a counter game where the object is to find pairs

When the game begins, all pictures are hidden

To play:

  1. Select two cards to try match the pictures.
  2. If you match the pictures you can go again.
  3. If they don't match it's the computer turn them.
  4. The player that find all pairs wins!
  5. To start the game, please enter your First name, Last name and email.
  6. Then press the Start game button.
  7. Have fun :)
